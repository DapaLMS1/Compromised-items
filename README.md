Compromised Item Detective Challenge

A clean, interactive web application designed for medical or laboratory training. This tool helps users identify and understand the common signs that a sterilized item or package has been compromised.

🌟 Overview

In sterile processing environments, recognizing a breach in sterilization is critical for patient safety. This application provides a "Detective Challenge" interface where users can click through various indicators—such as dampness, damaged packaging, crushed packs, and incorrect chemical indicators—to learn why these specific conditions render an item non-sterile.

🚀 Features

Interactive UI: Click-to-reveal tiles that provide detailed explanations.

Responsive Design: Fully functional on desktops, tablets, and mobile devices using a mobile-first approach.

Professional Aesthetics: Uses a sophisticated purple and green color palette suitable for healthcare and educational settings.

Micro-animations: Smooth hover effects and transitions to enhance user engagement.

🛠️ Technical Stack

HTML5: Semantic structure.

Tailwind CSS: Utility-first styling for layout and typography.

Vanilla JavaScript: Lightweight logic for state management and DOM manipulation.

CSS3 Custom Properties: Used for consistent branding and theme management.
